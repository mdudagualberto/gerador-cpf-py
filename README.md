# 🪪 Gerador de CPF

> Programa que gera automaticamente um CPF válido, seguindo as regras oficiais da Receita Federal!

---

## 📖 Sobre o Projeto

O **Gerador de CPF** cria números de CPF válidos automaticamente. O programa segue o algoritmo oficial utilizado pela Receita Federal do Brasil para calcular os dois dígitos verificadores — ou seja, os CPFs gerados passam em qualquer validação.

> ⚠️ **Aviso:** Este projeto tem fins exclusivamente educacionais. CPFs gerados não pertencem a nenhuma pessoa real e não devem ser usados para qualquer fim ilegal.

---

## 🎮 Como Funciona (Exemplo)

```
Gerando CPF...

CPF gerado: 123.456.789-09
```

O programa gera os 9 primeiros dígitos aleatoriamente e então calcula os 2 dígitos verificadores finais usando o algoritmo oficial do CPF.

---

## 🧮 Como o CPF é Validado?

Um CPF tem 11 dígitos no formato `XXX.XXX.XXX-DD`, onde os dois últimos (`DD`) são chamados de **dígitos verificadores**. Eles são calculados a partir dos 9 primeiros dígitos usando uma fórmula matemática — é exatamente isso que o programa faz!

## 🚀 Como Rodar

### Pré-requisitos

Você precisa ter o **Python** instalado na sua máquina.

> Não tem Python? Baixe em: https://www.python.org/downloads/

Para verificar se já está instalado, abra o terminal e digite:

```bash
python --version
```

Se aparecer algo como `Python 3.x.x`, está tudo certo!

---

### Passo a Passo

**1. Clone o repositório:**

```bash
git clone https://github.com/mdudagualberto/gerador-cpf-py.git
```

**2. Entre na pasta do projeto:**

```bash
cd gerador-cpf-py
```

**3. Execute o programa:**

```bash
python gerador_cpf.py
```
